# Mindmonk — landing page

Static landing page for **Mindmonk** ([@Mindmonk_opusbot](https://t.me/Mindmonk_opusbot)) — the bot that follows your favorite podcast channels, reads each new episode, and sends a sharp four-part digest to your Telegram.

- **Cinematic hero** — an aerial drone clip (`hero.mp4`) plays over a pure CSS/SVG dawn-mountains scene, with a slow aerial zoom. If the video fails to load, the animated scene is the fallback.
- **Minimal editorial copy** — large serif headline, a three-step "how it works", and the four-part digest card.
- **No build step.** Just open `index.html`, or deploy the folder to Netlify / Vercel / GitHub Pages.

## Swap the hero video
Replace `hero.mp4` with any aerial/drone clip (free, license-clear sources: [pexels.com/videos](https://www.pexels.com/videos/), [coverr.co](https://coverr.co), [mixkit.co](https://mixkit.co)). It fades in automatically once it can play.

Current clip: drone view of a serene sunrise over a misty river and mountains — Pexels, free / no attribution required.
